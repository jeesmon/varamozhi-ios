Setup

  1. Open Device Settings from Home screen
  2. Tap General -> Keyboard -> Keyboards -> New Keyboard
  3. Select Varamozhi from THIRD-PARTY KEYBOARDS
  4. Optional: Tap Varamozhi and tap Allow Full Access

Usage

  1. Tap and hold Globe icon on default keyboard and select മലയാളം (QWERTY)


  2. Type in Manglish and text will be converted to Malayalam
  3. Tap Globe icon on Varamozhi keyboard to switch back to other keyboards
